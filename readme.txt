
#Foram inseridos no repo os arquivos solicitados nas "Orientacoes para a avaliação final", 
de Andrea Sánchez-Tapia, Sara Mortara, Diogo S.B: Rocha, de 7/22/2019

A avaliação da disciplina será realizada sobre o repositório final de cada um. Como conversamos durante
a disciplina, estamos esperando um repositório por aluno, com uma estrutura de pastas interna que seja
coerente. Não precisa ser exatamente a mesma estrutura mas alguns elementos chave são:
• Uma pasta de dados onde seja possível distinguir os dados brutos e processados
• A pasta R com os scripts, em formato .R - Lembre das dicas para organização individual dos scripts
• Uma pasta de imagens onde sejam salvas as figuras criadas durante a execução dos scripts
Se tiver mais elementos ou precisar incluir qualquer coisa no arquivo .gitignore, fique à vontade.
A história de commits não será avaliada, apenas a versão mais recente. Não se preocupe em errar e corrigir
erros ao longo do caminho, o objetivo é que os scripts finais fiquem rodando de uma maneira reproduzível
• Comente muito seu código e inclua nos comentários quando surgir uma dúvida ou precisar explicar
algum passo
Os scripts que devem ser incluídos na pasta de scripts são os das
aulas:
• o exercício de formatação de dados (fazendo download do arquivo .xlsx e corrigindo os dados)
• 04 de manipulação de dados
• 06 do tutorial de análise exploratória de dados
• o exercicío de análise exploratória dos dados
• 07 da aula de gráficos
• o exercício da aula de gráficos (iris) e o desafio (opcional)
• 08 do tutorial de mapas
Dicas gerais:
• Todos os dados que vocês usem durante os exercícios devem estar na pasta ./data do repositório
• Todos os scripts que vocês escrevam durante os exercícios devem estar na pasta ./R do repositório
• Todas as imagens que vocês criem durante os exercícios devem estar na pasta ./figs do repositório
etc.
É opcional criar arquivos de markdown com estes resultados
Um repositório com uma estrutura de exemplo pode ser encontrado aqui
O prazo para o último push do repositório é dia 9 de agosto.
Bom trabalho! Qualquer dúvida estamos à disposição.
